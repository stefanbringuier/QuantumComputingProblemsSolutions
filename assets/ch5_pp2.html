Problem 2. Consider a 6×6 matrix A=aij, then calculate the partial trace in the basis:

(10)⊗I3,(01)⊗I3.

Also calculate the partial trace with the basis:

(100)⊗I2,(010)⊗I2,(001)⊗I2

11.5 μs

Solution. The key thing to recall is ta the partial trace defines an operator of a subspace so:

A:=trB(A⊗B)≡∑i=1NB(IA⊗⟨ϕi|)(A⊗B)(IA⊗|ϕi⟩)

where |ϕi⟩ define an orthonormal basis in B subspace.
